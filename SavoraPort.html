Savora — Your AI Kitchen Companion

Savora is a full-featured AI-powered cooking app designed to help home cooks discover, create, and perfect recipes using ingredients they already have. With a curated library of over 10,000 recipes and an intelligent AI chef assistant, Savora transforms everyday cooking into an effortless, creative experience.

The Problem

Most recipe apps either overwhelm users with too many options or fail to account for what's actually in their kitchen. Savora bridges this gap by combining a massive recipe database with smart pantry management and AI-driven recipe generation — so you always know what you can cook right now.

Key Features

Smart Pantry Management: Add ingredients you have on hand, and Savora instantly filters recipes you can make. The pantry syncs across devices and updates in real time.

AI Recipe Generation: Powered by OpenAI's GPT-4o-mini (and enhanced by a custom fine-tuned culinary model — Savora AI), the app generates personalized recipes based on your available ingredients, dietary preferences, and cuisine style.

10,000+ Curated Recipes: A professionally curated recipe database spanning global cuisines, with detailed nutritional information, step-by-step instructions, cook times, and difficulty ratings.

Personalized Experience: Dietary filters (vegetarian, vegan, gluten-free, keto, etc.), cuisine preferences, serving size adjustments, and a favorites system that learns your taste over time.

Cooking Tools: Built-in timers, unit converters, and step-by-step cooking mode that guides you through each recipe hands-free.

Tech Stack

Mobile App: React Native with Expo — cross-platform iOS and Android from a single codebase. Features include push notifications, camera integration for ingredient scanning, and offline recipe caching.

Web Platform: Next.js (React) with server-side rendering for the web companion app and marketing site.

Backend & API: Next.js API routes handling authentication, recipe search, pantry management, and AI integration. PostgreSQL database with Prisma ORM for structured recipe and user data.

AI Integration: OpenAI GPT-4o-mini for recipe generation and culinary Q&A, with a custom fine-tuned model (Savora AI) trained on 37,000+ culinary examples for expert-level cooking intelligence.

Infrastructure: Vercel for web deployment, Expo EAS for mobile builds and OTA updates, Supabase for real-time data sync and authentication.

Architecture Highlights

Savora uses a monorepo structure with shared business logic between the mobile and web platforms. The recipe recommendation engine combines collaborative filtering with the AI model to surface personalized suggestions. The pantry-to-recipe matching system uses an efficient ingredient intersection algorithm that works in real time even across 10,000+ recipes.
